What is Constructor

A constructor is a special method that is used to initialize objects. The advantage of a constructor, is that it is called when an object of a class is created. It can be used to set initial values for fields:

- has the same name as that of the class

- does not have any return type

Types of Constructors:

- Default constructor : it's automatically provided by the compiler if you don't define any other constructors. The default constructor typically initializes all instance variables to their default values .

- Parameter constructor: You can define constructors with parameters to allow you to specify the initial values for the object's properties during creation. This provides more control over the object's initial state.

              class Car {

- Lets take a Example of Dafualt constructor: 

            
              // Create a Car class
              class Car
              {
                public string model;  // Create a field
              
                // Create a class constructor for the Car class
                public Car()
                {
                  model = "Mustang"; // Set the initial value for model
                }
              
                static void Main(string[] args)
                {
                  Car Ford = new Car();  // Create an object of the Car Class (this will call the constructor)
                  Console.WriteLine(Ford.model);  // Print the value of model
                }
              }
              
              // Outputs "Mustang"
